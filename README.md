# react-scaffold

Basic react scaffold to build admin app.

## Usage

```shell
yarn install
yarn dev
yarn build
```

## Lint

### VSCode

Install `eslint`, `prettier` plugins.
Add config in `settings.json`:

```json
{
  "files.autoSave": "onFocusChange",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  }
}
```
